# COVID-19 Data Analysis

An exploratory data analysis (EDA) project on COVID-19 datasets using **Python**, **Pandas**, **Matplotlib**, and **Seaborn** in a Jupyter Notebook. This project visualizes global COVID-19 trends including confirmed cases, recoveries, and deaths.

## Project Overview

This notebook performs data cleaning, processing, and visualization of COVID-19 data to uncover patterns and insights about the pandemic's spread across countries and over time.

## Features

- Data loading and preprocessing with **Pandas**
- Visualizations of global case trends (confirmed, recovered, deaths)
- Country-wise analysis and comparison
- Time-series plotting using **Matplotlib** / **Seaborn**
- Dataset stored locally in the `datasets/` folder

## Tech Stack

| Tool | Purpose |
|---|---|
| Python 3 | Core programming language |
| Jupyter Notebook | Interactive analysis environment |
| Pandas | Data manipulation and cleaning |
| Matplotlib | Data visualization |
| Seaborn | Statistical data visualization |

## Getting Started

1. Clone the repo:
   ```bash
   git clone https://github.com/shreyawritescode/covid_data_analysis.git
   cd covid_data_analysis
   ```
2. Install dependencies:
   ```bash
   pip install pandas matplotlib seaborn jupyter
   ```
3. Launch Jupyter Notebook:
   ```bash
   jupyter notebook notebook.ipynb
   ```
